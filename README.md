# Description
`cutpointpy` is a Python package for estimating the optimal cut-point of a predictor variable (feature) for a binary classification task. It is loosely inspired on [`cutpointr`](https://cran.r-project.org/web/packages/cutpointr/index.html), an optimal cut-point calculation package for [R](https://www.r-project.org/). In particular, `cutpointpy` allows the following:

 - Optimal cut-point estimation
 - Stability analysis of the estimated cut-points through bootstrapping
 - Receiver-operating characteristic (ROC) analysis
